# Changelog

All notable changes to this project will be documented in this file.

## [1.0.2] - 2026-01-25

### Fixed
- Fixed logout redirection issue causing white screen when using default settings.
- Fixed character encoding syntax error in package script.

## [1.0.1] - 2026-01-25

### Added
- Dynamic "Primary Color" customization setting.
- Shared CSS system for consistent styling across blocks.
- `accent-color` support for checkboxes.

### Fixed
- Fixed URL redirector stripping query parameters (reset key).
- Fixed missing button hover styles on reset form.
- Fixed legacy admin page registration.

## [1.0.0] - 2026-01-25 

### Added
- Initial release
- PSR-4 autoloading structure
- Example REST API controller
- Example admin page
- Example Gutenberg block
- Database schema management
- Activation/deactivation hooks
- Internationalization support
- Security best practices
- Extensibility hooks and filters
