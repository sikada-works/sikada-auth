# Customization

This tab allows you to style your login flow and customize text labels **without writing any code**. Changes made here update instantly across your entire site.

## Design

### Primary Color
*   **What it does:** Sets the main brand color for your forms.
*   **Affects:** Login buttons, link colors, and input focus rings (the glow when you click a box).
*   **How to use:** Click the color box to choose a color from the picker, or paste your brand's Hex code (e.g., `#FF5733`).
*   **Default:** WordPress Blue (`#2271b1`).

## Login Form
Customizes the "Login" block.

*   **Username Label:** The text above the first box. (Default: "Username or Email")
*   **Password Label:** The text above the password box. (Default: "Password")
*   **Remember Me Label:** The text next to the checkbox. (Default: "Remember Me")
*   **Login Button:** The text inside the blue submit button. (e.g., Change to "Sign In" or "Access Portal")
*   **Lost Password Link:** The text for the help link at the bottom. (Default: "Lost your password?")

## Password Reset Form
Customizes the "Password Reset" block.

*   **Email Request Label:** The label for the input when a user asks for a reset link.
*   **Request Button:** The button text to send the email. (Default: "Get New Password")
*   **New Password Label:** The label for the first box when setting a new password.
*   **Confirm Password Label:** The label for the second box to verify the password.
*   **Reset Button:** The button text that saves the new password. (Default: "Reset Password")

> **Note:** If you leave any of these fields blank and save, they will revert to their default English values.
